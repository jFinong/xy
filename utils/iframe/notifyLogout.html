    # 登录超时通知 iframe

    iframe 内的子应用登录超时通知 iframe 登出。

    # 在 post.js 中超时判断处添加代码：

    ...
if (window.self !== window.parent) {
    window.parent.postMessage(
        {
            logout: true,
        },
        '*'
    );
    return false;
}
